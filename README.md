Aplikasi task manager berbasis file JSON yang berjalan di command-line

Fitur:
- Tambah Tugas: Menambahkan tugas baru ke daftar.
- Lihat Daftar Tugas: Menampilkan semua tugas dengan status (selesai/belum).
- Tandai Selesai: Mengubah status tugas menjadi selesai.
- Hapus Tugas: Menghapus tugas dari daftar

Cara Penggunaan:
- Tambah Task => node index.js add "Tugas CLI"
- Lihat Semua Task => node index.js list
- Tandai Task Selesai => node index.js done 1
- Hapus Task => node index.js remove 1
